ID e CLASS

ID

O id funciona como um CPF. assim como só posso ter um numero de CPF,cada elemento só pode ter um valor de ID. o CPF é unico,ou seja,ninguem pode ter um CPF igual ao meu.isso que o ID tambem é unico, antão não é possovel repetir o mesmo valor duas vezes.

CLASS

o CLASS funciona de maneira oposta ao ID. diferente do ID em que voce pode ter um valor, no CLASS voce pode ter varios valores. tambem diferente do ID, em que não se pode repetir um valoe, no CLASS voce pode utilizar o mesmo valor para varios elementos.
